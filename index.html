Summary: Latitude and Weather

The analysis presented in this dashboard explores the relationship a city's proximity to the equator has on the weather conditions of that city. A python script was developed in order to conduct this analysis. By utilizing the citipy library and the OpenWeatherMap API within that python script, a representative model of weather for a dataset of over 500 cities was created.

Once the dataset was assembled and all the API calls had been made, the results were then plotted using Matplotlib. The plots were used to analyze the cities' latitude against four distinct weather components: Temperature, Humitidy, Cloudiness, and Wind Speed. Within the pages of this site you will find visualizations of the plots that were created, descriptions/explanations of those plots, a page that shows all four plots together for comparison, and a page containing all of the data that was used to create the plots.
